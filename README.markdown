=========
Originally hacked up by Jake McGinty

Normal Commands
---------------
*NOTE*: All commands can be sent in either the channel internbot is residing in or via private message (`/msg internbot <cmd>` or `/query internbot` to open a chat)

* WHOIS-type lookups
        
        whois <irc user>
        face <irc user>

* Gnarly

        bro me
        add bro <bro name>
